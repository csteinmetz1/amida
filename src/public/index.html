amida

audio mixing interface for data acquisition

amida is a web interface for collecting data on how audio engineers mix. After entering some background information, users are presented with a virtual mixing console with four channels. Users can tweak the mix by adjusting the level of these channels until they are satisfied with the mix. When finished with a mix, users MUST press the Save button. This will store the user's mix to the server and mark that song as completed.


When you are ready press the Begin button below.
