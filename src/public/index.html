amida

audio mixing interface for data acquisition

amida is a web interface for collecting data on how audio engineers mix. After entering some background information, you will be presented with a virtual mixing console with four channels. You are free to adjust the level of these channels until you are satisfied with the mix. When finished with a mix, you must press the Submit button. If you feel you cannot achieve a good mix, you can use the Skip button. When you have mixed as many songs as you would like press the End button. For more direction see the Details page.
