amida

audio mixing interface for data acquisition

Click New User if this is your first time using amida.

Click Login if you are returning to a session.



about

amida is a web interface for collecting data on how audio engineers mix. After logging in users are presented with a virtual mixing console with four channels. Users can tweak the mix by adjusting the level of these channels util they are satisfied with the mix. When finished, pressing the Save button will store the users mix to the server and will mark that song as completed.
